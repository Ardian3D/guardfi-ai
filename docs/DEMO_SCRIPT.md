# GuardFi AI — Demo Script (3–5 minutes)

> Keep it calm and honest. GuardFi AI is **automated risk intelligence, not a formal audit or financial advice.**

## 0. Setup before you present
- `apps/web/.env.local` has `NEXT_PUBLIC_REGISTRY_ADDRESS=0xBa0422Df609cA9f0E6157Da62075aa12316801Ea` and `NEXT_PUBLIC_HSP_MODE=mock`.
- Wallet (MetaMask) installed, funded with a little HSK for the on-chain step.
- App running: `cd apps/web && npm run dev` → http://localhost:3000
- A target contract address ready (or use the Sample Target demo chip).

---

## 1. Opening pitch (~20s)
"Before you interact with any DeFi contract, you're trusting code you can't read. GuardFi AI scans a contract on HashKey Chain, scores its risk deterministically, explains it with AI, and anchors a tamper-evident proof on-chain that anyone can verify."

## 2. Problem (~20s)
"Hidden owner privileges, unlimited minting, blacklists, upgradeable proxies, unverified source — these are invisible to most users until it's too late. Audits are slow, and reports are hard to trust."

## 3. Product explanation (~20s)
"GuardFi AI is four things in one flow: a deterministic risk engine, an AI explainer, an on-chain proof registry, and a public verifier. The score is reproducible; the AI only explains what the engine found."

## 4. Live demo flow (~2–3 min)
1. **Landing page** → click **Launch App**.
2. **Connect wallet** → approve → **switch to HashKey Chain**.
3. Land on the **Dashboard** (empty state on first run — "No activity yet").
4. Click **New Scan** → paste a **target contract** (or click **Sample Target (demo)**).
5. Click **Analyze Risk** → the app reads bytecode + ERC-20 metadata + proxy signal + selectors, then shows a **risk score (0–100)**, level, and indicators.
6. Click **Generate AI Report**.
   - "This is DeepSeek explaining the deterministic findings. Notice the source badge: **Generated by DeepSeek**, or **Fallback Report** if no key is configured — it never breaks."
7. Click **Unlock Premium Report** → the HSP modal opens with the **Mock HSP Mode** badge → **Unlock in Mock Mode** → status goes Created → Pending → Settled.
   - "Payment is behind an adapter. This is clearly mock mode — no real value moves — and unlocking does **not** change the on-chain hash."
8. Click **Save Proof On-Chain** → confirm the transaction in the wallet.
   - "My wallet signs it. The backend never submits. Only a minimal commitment goes on-chain."
9. Open the **transaction on HashKey Blockscout** (`https://hashkey.blockscout.com`).
10. Go to **/verify**, enter **Report ID `1`** → the on-chain reporter, target, score, report hash, metadata URI, and timestamp appear.
    - "This page is fully public — no wallet needed. Anyone can verify a report exists and matches."
11. Return to **Dashboard** → show the new scan, AI report, on-chain proof, and premium unlock reflected in the activity.

## 5. AI report explanation (~15s)
"The AI is constrained: it only receives the structured scan result and is told not to invent data, not to give financial advice, and not to claim it's an audit. Output is strict JSON, validated and sanitized."

## 6. On-chain proof explanation (~15s)
"We hash a deterministic report payload with keccak256 and store only that hash plus metadata in `GuardFiReportRegistry`. The full report stays off-chain — cheap, private, and still verifiable."

## 7. Verification explanation (~10s)
"Verification is read-only and public: look up the report by ID, compare the on-chain hash to a known report hash. No wallet, no payment."

## 8. HSP premium unlock explanation (~10s)
"Scanning and verification are free. The full AI report is premium, settled with HSP through a swappable adapter. Today it's mock mode for the demo; the real adapter is a documented skeleton."

## 9. Dashboard explanation (~10s)
"The dashboard aggregates your local activity — scans, AI reports, on-chain proofs, and premium unlocks — with per-wallet unlock filtering."

## 10. Closing pitch (~15s)
"GuardFi AI makes DeFi risk legible and verifiable on HashKey Chain: deterministic scoring, responsible AI, and a public on-chain proof layer. It's an MVP — honest about what's real and what's mock — but the full flow works end to end today."
